# User Registration Web Application

A responsive full-stack web application built using Node.js, Express.js, EJS, JavaScript, Bootstrap, HTML, and CSS. The project features user registration, real-time validation, dynamic DOM manipulation, responsive design, and server-side form processing.

## Features
- User Registration Form
- Client-side & Server-side Validation
- Live Password Strength Checker
- Real-time Phone Validation
- Dynamic DOM Manipulation
- Responsive UI with Bootstrap
- CSS Animations and Hover Effects
- EJS Templating

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Node.js
- Express.js
- EJS

## Learning Outcomes
- Frontend & Backend Integration
- Form Handling and Validation
- DOM Manipulation
- Express Routing
- EJS Rendering
- Responsive Web Design
